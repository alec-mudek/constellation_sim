# constellation_sim v0.1
Basic modeling and simulation environment for a Walker-Delta constellation about a planetary body.

## Dependencies
Eigen: https://libeigen.gitlab.io/
cspice: https://naif.jpl.nasa.gov/naif/toolkit_C.html
astrokit: https://github.com/alec-mudek/astrokit/

A basic CMakeLists.txt file will be added shortly.
